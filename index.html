Drop-in Web Components

Simple, Hosted components.

See an example

Easy to use

Drop-in web components require very little configuration. In most cases, all you need is a single element.

 

Built on jQuery

Drop-in web components are designed to package existing jQuery plugins as easy-to-install, hosted components.

 

Client-Side JavaScript

Drop-in web components are written in client-side JavaScript, making them fast and flexible in any modern browser.

 

Easy to extend

We aim to make it as painless as possible to build new components, or package your existing jQuery plugin as a component.
